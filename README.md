[작업 규칙]
- 모든 작업은 main이 아닌 자신의 작업 브랜치에서 수행한다.
- 자신의 이름-작업내용 폴더를 만들고, 다른 팀원의 작업물은 수정하지 않는다.
- 충돌 테스트는 팀원 전체 동의 후 수행하며, 복구 커밋은 명확히 남긴다.
- 각자 폴더의 문서 작성은 README.md 하나로 통일한다.
